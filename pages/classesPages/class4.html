For

In JavaScript, the for loop is used to execute a block of code multiple times based on a condition. It consists of three parts: initialization, condition, and increment/decrement. It is commonly used for iterating over arrays or executing a specific number of repetitions.

for( let i = 10; i > 0; i-- ){

console.log('number', i)

}

While

In JavaScript, the while loop repeatedly executes a block of code as long as a specified condition remains true. It is useful when the number of iterations is not fixed and depends on a dynamic condition.

let cont = 100

while(cont > 0 ){

console.log("cont: ", cont)

cont = cont - 10

}

Do While

In JavaScript, the do...while loop executes a block of code at least once, regardless of the condition. After the first execution, it continues looping as long as the condition remains true. This makes it useful when the code needs to run before checking a condition.

let a = 0;

do{

if( a % 2 == 0 ){

document.write( a )

}

a++

}while(a <= 20)

Switch Case

In JavaScript, the switch statement is used to execute one of many blocks of code based on different possible values of a variable or expression. It is an alternative to multiple if...else statements and is particularly useful when dealing with many conditions.

let operations = Number(prompt("choose you operation | addition = 1 | subtraction = 2 | multiplication = 3 | division = 4|"))

let num1 = Number(prompt("Enter with the first number: "))

let num2 = Number(prompt("Enter with the second number: "))

switch(operations){

case 1:

addition = num1 + num2

document.write(`Addition result = ${addition}`)

break;

case 2:

subtraction = num1 - num2

document.write(`Subtraction result = ${subtraction}`)

break;

case 3:

multiplication = num1 * num2

document.write(`multiplication result = ${multiplication}`)

break;

case 4:

division = num1 / num2

document.write(`Division result = ${division}`)

break;

default:

document.write()

}
