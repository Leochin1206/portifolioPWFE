Dialog Boxes

Dialog boxes are pop-up windows that appear on the screen to display messages or interact with the user. They are useful for simple notifications, confirmations, and data input, but they are not recommended for more sophisticated interfaces as they block code execution until the user interacts with them. There are three types:

Alert -> Displays a dialog box with a message and an "OK" button. It is used to inform the user.

alert("Hello World!");

Confirm -> Displays a dialog box with a message and "OK" and "Cancel" buttons. Returns true if the user clicks "OK" and false if they click "Cancel."

confirm("Do you like JavaScript?");

Prompt -> Displays a dialog box with a message and an input field for the user to type something. Returns the entered value or null if the user cancels.

let name = prompt("Hello World!");
console.log(name);

Variables

var:

- Has function scope.

- Can be redeclared within the same scope.

- Can cause issues due to hoisting.

var name = "João";
var name = "Maria";
console.log(name);

let:

- Recommended for mutable variables.

- Has block scope ({}), avoiding variable conflicts.

- Can be updated, but not redeclared within the same scope.

let age = 25;
age = 40;
// let age = 40; // Erro
console.log(age);

const:

- For constant values.

- Also has block scope.

- Cannot be reassigned after declaration.

var name = "João";
var name = "Maria";
console.log(name);

Writing a phrase in the HTML body

The document.write() command inserts text directly into the HTML body, displaying the value placed inside the parentheses. However, its use should be cautious, as it can replace all the content of the page.

let subject = "Math";
document.write(`My favorite subject is ${subject}!`)
