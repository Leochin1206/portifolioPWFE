Functions

A function in JavaScript is a reusable block of code that performs a specific task. It can accept parameters and return a value. Functions are defined using the function keyword.

function greet(name) {

return "Hello, " + name;

}

console.log(greet("John")); // Output: "Hello, John"

Arrow Function

Arrow functions are a shorter syntax for writing functions. They are defined using the = > operator. Arrow functions do not have their own this context and are often used for shorter functions or callbacks.

const greet = (name) => "Hello, " + name;

console.log(greet("John")); // Output: "Hello, John"

Changing HTML using JS

In JavaScript, you can dynamically modify HTML elements by selecting them using methods like getElementById(), getElementsByTagName(), and querySelector(), and then changing their properties, such as color, background, or size, through the style attribute. These changes can be triggered by user interactions, like clicking a button, and conditional logic can be used to alter styles based on the current state. This ability makes web pages interactive and responsive, improving the user experience.

description 1

let h1 = window.document.getElementsByTagName('h1')[0];

h1.style.color='red';

description 2

let body = window.document.body;

body.style.background="green";

description 3

let box = document.querySelector(".box");

box.style.backgroundColor = "blue";

box.style.width ="800px";
