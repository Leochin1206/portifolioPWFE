Functions

A function in JavaScript is a reusable block of code that performs a specific task. It can accept parameters and return a value. Functions are defined using the function keyword.

function greet(name) {

return "Hello, " + name;

}

console.log(greet("John")); // Output: "Hello, John"

Arrow Function

Arrow functions are a shorter syntax for writing functions. They are defined using the = > operator. Arrow functions do not have their own this context and are often used for shorter functions or callbacks.

const greet = (name) => "Hello, " + name;

console.log(greet("John")); // Output: "Hello, John"

Changing HTML using JS

In JavaScript, you can dynamically modify HTML elements by selecting them using methods like getElementById(), getElementsByTagName(), and querySelector(), and then changing their properties, such as color, background, or size, through the style attribute. These changes can be triggered by user interactions, like clicking a button, and conditional logic can be used to alter styles based on the current state. This ability makes web pages interactive and responsive, improving the user experience.

This code selects the first h1 element on the page using getElementsByTagName('h1')[0] and stores it in the h1 variable. Then, it modifies the text color of the selected h1 element by setting its style.color property to 'red', changing the color of the text to red.

let h1 = window.document.getElementsByTagName('h1')[0];

h1.style.color='red';

This code selects the "body" element of the document using "window . document . body" and stores it in the "body" variable. Then, it changes the background color of the entire page by setting the "style . background" property of the "body" element to "green", making the background color green.

let body = window.document.body;

body.style.background="green";

This code selects the first HTML element with the class .box using document.querySelector(".box") and stores it in the box variable. It then modifies the selected element by setting its background color to blue (box.style.backgroundColor = "blue") and adjusting its width to 800 pixels (box.style.width = "800px"). As a result, the element with the class box will have a blue background and a width of 800px.

let box = document.querySelector(".box");

box.style.backgroundColor = "blue";

box.style.width = "800px";

This code defines a function called "mudarCor" that changes the background color of a "div" element with the ID "minhaDiv". When the function is called, it checks if the current background color of the "div" is "fuchsia" using the condition "div.style.background == "fuchsia". If the color is "fuchsia", it changes the background color to "violet". If the color is not "fuchsia", it sets the background color to "fuchsia". Essentially, this function toggles the background color between "fuchsia" and "violet" each time it's executed.

function mudarCor() {

let div = document.getElementById("minhaDiv");

if(div.style.background == "fuchsia"){

div.style.background = "violet"

}else{

div.style.background = "fuchsia"

}

}
